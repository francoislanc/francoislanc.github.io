The model, named Midistral, is finetuned on the MidiCaps dataset. The MIDI files from the dataset are converted to ABC notation files using midi2abc executables.

The input is a text description with genre, mood, and instruments.
The output is ABC music notation that can be converted to MIDI and audio files.

The source code is available on Github.
