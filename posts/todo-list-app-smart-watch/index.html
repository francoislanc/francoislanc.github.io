TodoList app for a smartwatch ⌚.
Posted on 2022-03

TodoList is a simple app for the Bangle.js smartwatch to manage a list of todo items.

The Bangle.js uses an open-source JavaScript interpreter for microcontrollers: Espruino. It is designed for devices with small amounts of RAM (as low as 8kB).

Developed with Espruino.
The source code is available on Github.
